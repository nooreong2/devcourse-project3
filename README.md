# KBO 야구 경기장 대시보드를 위한 Airflow DAGs

## 프로젝트 주제
- KBO 경기장 별로 경기 당일의 기상 상태를 확인하여 경기 취소 가능성을 시각적으로 확인할 수 있는 대시보드 제작
- 각 경기장 별 경기일정 데이터 수집 → 기상청 API 데이터를 활용한 우천/폭염/미세먼지/풍속 4가지 경기취소 조건별 시각화
